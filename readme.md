Este sketch corresponde al proyecto numero 3 para la asignatura de
Imagen escrita del año 2015. Pueden ver este y otros trabajos
realizados por otros compañeros aqui:

http://imagen-escrita.hspencer.info/

El nombre del sketch hace referencia a la idea del presente
sketch (dibujar con la voz) y una ironía del mundo del sonido
(la nota FA sostenido no existe). 

QUE ES LO QUE HACE?
=================================================================
La idea del sketch es dibujar en pantalla con la voz. Uno puede
cantar una canción que desee, o bien jugar con la propia voz u
otros sonidos para que el sketch dibuje formas con circulos de
diferentes tamaños y colores, dependiendo de la tonalidad y el 
volumen en entonces.  Uno puede dibujar, así como tambien
borrar elementos ya dibujados en la pantalla, así como tambien
empezar otra vez si el resultado no es como uno deseaba.
